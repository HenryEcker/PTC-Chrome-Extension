Wrong File Name Feedback Template

The error message you received:

File Name (0.0/1.0)
File "main.cpp" not found! Please make sure your file is named correctly.

Means that your file was supposed to be named "main.cpp" and it was not. Incorrectly named files become lost or de-referenced in large systems. The file name is just as important as its contents. Additionally, the error is easy to fix. When you see this error message, simply rename your file to the correct value and resubmit.
